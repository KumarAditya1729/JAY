# JAY Website Constitution

## Mission:
Never generate template-quality websites.

Every website must appear as if it was designed by a senior Silicon Valley product team.

## Requirements:

1. Design must be custom and brand-specific.
2. Typography must be intentional and premium.
3. Layout must create visual hierarchy and storytelling.
4. Mobile experience is mandatory.
5. Performance target is Lighthouse 90+.
6. Every page must have a clear business objective.
7. Generic Tailwind SaaS templates are forbidden.
8. Every design must be benchmarked against Stripe, Linear, Apple, Vercel, Notion, and other world-class products.
9. Before approval, run Design Review, UX Review, Engineering Review, Conversion Review, and Brand Review.
10. If the result looks like a freelance template, reject and rebuild automatically.

## Success Metric:
A founder should be proud to show the website to investors, enterprise clients, and Fortune 500 executives.
